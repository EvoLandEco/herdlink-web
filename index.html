Quick Start
Select a time resolution from the top-left switches.
: daily / : weekly / : monthly / : yearly
Toggle Graph / Map view using the first button at the bottom-right of the network panel.
: Graph view / : Map view / or press M to switch.
Click a node to enter focus mode; press Q to exit.
Use / to switch focal nodes quickly; see the side panels for detailed stats and trade info.
If time controls are available, press Space to play / pause, / to step time, F to jump to start.
In focus mode, links can be turned off in the right panel. Press R to restore links and S to export a screenshot.
Press Esc to quickly exit the help overlay. Press H anytime to toggle it.
Shortcuts
S Screenshot
Export the current view as PNG.
M Map / Graph
Toggle between map and network views.
Q Exit focus
Quit focus mode for the selected node.
R Restore links
Re-enable all links after filtering.
H Help
Toggle the help overlay.
Space Play / pause
Toggle time replay.
F From start
Jump back to the first time step.
←→ / ↑↓
Time step / Switch node.
